Q2. You are given a string (STR) of lenght N. you have to print the count of all alphabet.(using maps)

                input: 
                    abcadeecfb

                output: 
                a=2 
                b=2 
                c=2 
                d=1 
                e=2 
                f=1 
                
                ans: 
                var str = "abcadeecfb";
                const myMap = new Map();
                for (var alphabet of str) {
                if (myMap.has(alphabet)) {
                    myMap.set(alphabet, myMap.get(alphabet) + 1);
                } else {
                    myMap.set(alphabet, 1);
                 }
                }
                console.log(myMap);
